# Creator Thread / Long-form Template

★ ใช้กับ creator type สำหรับ:
- Twitter/X thread
- FB long-form post (500+ คำ)
- IG carousel script (1 idea per slide)
- LinkedIn long post

## Structure (10-12 ทวีต / slide / paragraph)

### Slot 1 — Hook (1 unit)
- Misconception / question / counter-intuitive
- ระบุ value upfront — "ใน thread นี้ คุณจะเข้าใจ X ใน 2 นาที"

### Slot 2 — Why this matters (1 unit)
- ทำไมเรื่องนี้สำคัญตอนนี้
- ใครควรอ่าน

### Slots 3-8 — Substance (5-6 units)
- 1 idea per slot
- จัดเรียงเป็น flow ชัด:
  - Setup → Frame → Examples → Implications
  - หรือ Problem → Cause → Solution → Tradeoffs → Path forward
- ใส่ตัวเลข/framework/quote ที่ตรง slot นั้น

### Slot 9 — Counter-perspective (1 unit) ★
ที่ creator content ที่ดีต่างจาก generic ตรงนี้:
- ยอมรับ "but / however / there's a catch"
- แสดงความ humble / acknowledge complexity

ตัวอย่าง: "ทั้งนี้ X อาจไม่ work ในกรณี Y — สิ่งที่ผมพบคือ..."

### Slot 10 — Takeaway (1 unit)
- 1-3 main points สรุป
- พูดสิ่งเดียวที่ผู้อ่านควรจำ

### Slot 11 — CTA + Soft Outreach (1 unit)
- ขอ engagement: comment / quote / share
- ถ้ามี course/service link — ใส่ตอนนี้แต่ soft

### Slot 12 — Optional: Self-promotion (1 unit)
ถ้าเป็น promo post ของ hybrid creator — เพิ่ม slot นี้ ไม่งั้น skip

## IG Carousel Adaptation

map slots → slides:
- Slide 1 = Hook (cover slide ที่กดให้คนเลื่อน)
- Slide 2 = Why this matters
- Slides 3-7 = Substance (1 idea per slide, ใช้ภาพ/diagram)
- Slide 8 = Counter-perspective (optional ถ้า scope ใหญ่)
- Slide 9 = Takeaway
- Slide 10 = CTA + brand tag

## Twitter/X Thread Adaptation

- Tweet 1 = Hook + "🧵" หรือ "↓"
- Tweet 2-N = 1 idea per tweet
- Last tweet = "ถ้าคุณชอบ thread นี้:" + ขอ retweet/follow
- ใส่ "1/", "2/", ... ทุก tweet เพื่อ reading order

## Length Target

| Format | Total length |
|--------|--------------|
| Twitter thread | 10-12 tweets |
| FB long-form | 500-800 คำ |
| IG carousel | 8-10 slides + caption 80 คำ |
| LinkedIn post | 600-1000 คำ |

## Quality Bar

- ✅ Each slot stands alone (มี 1 idea ที่จบในตัว)
- ✅ Flow ชัด — slot N → N+1 ควรเชื่อมโยง
- ✅ มี "aha moment" อย่างน้อย 1 ที่
- ❌ ไม่มี slot ที่เป็น filler — ตัดทิ้ง
- ❌ ไม่ pure listicle ที่ไม่มี perspective ของ creator
