# Structure by Business Type

รายละเอียด structure pattern แต่ละ type — ใช้ร่วมกับ template

---

## Product Type

### FB Post (medium 200-300 คำ)

| Section | Length | Purpose |
|---------|--------|---------|
| Hook | 1-2 บรรทัด | Pain question / quote จากคนจริง |
| Pain elaboration | 3-4 บรรทัด | สะท้อนความรู้สึก |
| Solution + soft mention | 5-7 บรรทัด | Value + ระบุสินค้า |
| CTA | 1-2 บรรทัด | "ทักแชท" / "ดูสินค้า" |

### Blog (800-1200 คำ)

| Section | Length |
|---------|--------|
| Title (≤60 char) + Intro | 100-150 คำ |
| H2: Pain context | 150-200 |
| H2: Why current solutions fall short | 150-200 |
| H2: Our approach + product | 200-300 |
| H2: How to use / get started | 150-200 |
| Outro + CTA | 100-150 |

## Creator Type ★

### FB Post (Educational, medium)

| Section | Length | Purpose |
|---------|--------|---------|
| Hook | 1-2 บรรทัด | Misconception / question / counter-intuitive |
| Setup | 1-2 บรรทัด | Context — ทำไมเรื่องนี้สำคัญ |
| Insight/Reveal | 3-5 บรรทัด | Substance + unique_perspective |
| Example | 1-2 บรรทัด | Case จริงจาก research |
| Takeaway + CTA | 1-2 บรรทัด | 1 thing to remember + save/follow/share |

### FB Long-form / Thread (500+ คำ)

ดู `templates/creator-thread.md` — 10-12 slot structure

### IG Carousel

| Slide | Purpose |
|-------|---------|
| 1 (Cover) | Hook ที่ทำให้กดเลื่อน |
| 2 | Why this matters |
| 3-7 | 1 idea per slide |
| 8 | Counter-perspective (optional) |
| 9 | Takeaway summary |
| 10 | CTA + brand tag |

Caption สั้น 80-120 คำ — สรุป 3 จุดเด็ด + CTA

### Blog (Educational, 1200-1800 คำ)

| Section | Length |
|---------|--------|
| Hook + Intro | 150-200 |
| H2: The Common Misconception | 250-350 |
| H2: What's Actually Going On | 350-500 |
| H2: Real Example / Case | 200-300 |
| H2: How to Apply This | 150-250 |
| Outro: 1 takeaway + CTA | 100-150 |

## Personal Brand Type

### FB Post

| Section | Length | Purpose |
|---------|--------|---------|
| Hook (personal angle) | 1-2 บรรทัด | "เคสที่เจอ..." / "หลังทำมา X ปี..." |
| Observation | 2-3 บรรทัด | สิ่งที่เห็นใน case/event |
| Analysis (use expertise) | 4-6 บรรทัด | วิเคราะห์ด้วย framework |
| Recommendation | 2-3 บรรทัด | actionable insight |
| CTA | 1-2 บรรทัด | DM / website |

### LinkedIn Long Post

| Section | Length |
|---------|--------|
| Hook (personal/news angle) | 100-150 คำ |
| Context / observation | 150-200 |
| Analysis | 250-350 |
| Recommendation | 150-200 |
| Reflection / CTA | 100 |

## CTA Pattern Summary

| Type | Primary | Secondary | Voice in CTA |
|------|---------|-----------|--------------|
| product | "ทักแชทมาคุย" / "สั่งเลย" | "ดูสินค้าทั้งหมด" | direct, sales-y |
| creator (non-promo) | "Save ไว้อ่าน" / "Follow" | "Share/Comment" | conversational, peer-like |
| creator (promo) | "DM ถามรายละเอียด" | "ดู course ที่ link" | soft, transparent |
| personal_brand | "DM ปรึกษา" | "ดู website" | professional |

---

## Hook Pattern Summary

| Type | Best hooks | Avoid |
|------|-----------|-------|
| product | Pain question, customer quote | Promo announcement |
| creator | Misconception, provocative Q, counter-intuitive | Listicle opener, summary |
| personal_brand | "เคสที่เจอ", news commentary, "หลังทำมา X ปี" | Generic motivational |

## Structure Variations by Goal

ถ้าผู้ใช้ระบุ goal ให้ปรับ structure:

### Goal: Engagement (comments)
- creator/personal_brand: เพิ่มคำถามท้ายโพสต์
- product: เปิดโอกาสคนเล่า pain ของตัวเอง

### Goal: Save/Share
- ใส่ value ที่ portable — checklist, framework, template
- ทำให้รูปภาพ/ข้อความ snippet-able

### Goal: Click-through (ไป link)
- product: CTA ตอน peak emotion (หลัง pain elaboration)
- creator: ห้าม link ในเนื้อหา — ใส่ comment/bio

### Goal: Follow growth
- creator: เน้น takeaway → "Follow เพื่อตอนต่อไป"
- ไม่ promote product
